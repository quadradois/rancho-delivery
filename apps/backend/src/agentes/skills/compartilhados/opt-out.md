# Skill: Opt-Out

## Objetivo
Lidar com clientes que pedem para parar de receber mensagens.

## Comportamento
Quando o cliente pedir para sair da lista ou parar de receber mensagens:

1. Respeite imediatamente o pedido — não tente convencer a continuar
2. Confirme de forma breve e amigável que não entrará mais em contato
3. Deixe aberta a possibilidade de retorno voluntário
4. **Não** envie mais mensagens após a confirmação

## Resposta padrão
"Tudo bem! Não vou mais te enviar mensagens. Se quiser nosso cardápio no futuro, é só chamar. Bom dia! 😊"

## Observação
Após detectar opt-out, o sistema registra o número para não contatar novamente automaticamente.
