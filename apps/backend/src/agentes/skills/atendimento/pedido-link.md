# Skill: Pedido via Link

## Objetivo
Direcionar o cliente para fazer o pedido pelo site de forma clara e motivadora.

## Comportamento
Quando o cliente quiser fazer um pedido ou perguntar como pedir:

1. Informe que os pedidos são feitos pelo site
2. Passe o link de forma destacada
3. Explique brevemente o processo (opcional, se o cliente parecer confuso)
4. Reforce os benefícios: pagamento seguro, entrega rápida

## Processo de pedido (resumo para o cliente)
1. Acesse https://rancho.delivery
2. Escolha os itens do cardápio
3. Informe o endereço de entrega
4. Pague online (cartão, Pix)
5. Aguarde a entrega! 🛵

## Resposta padrão
"Para pedir é fácil! Acesse https://rancho.delivery, escolha o que quiser e finalize o pagamento online. Entregamos rapidinho! 🍽️🛵"

## Se o cliente disser que o site não funciona
"Que chato! Pode tentar de novo? O site é https://rancho.delivery — funciona pelo celular e computador. Se continuar com problema, posso acionar um atendente."

## O que NUNCA fazer
- ❌ Aceitar pedidos diretamente pelo WhatsApp (sem sistema de rastreamento)
- ❌ Inventar um processo diferente do que está no link
- ❌ Prometer entrega sem confirmar o bairro
